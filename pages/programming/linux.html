Linux folder structure

Linux folder structure vary from version to version.

/dev/: The /dev directory contains the special device files for all the devices.

/etc/: CONFIGURATION FILES. The meaning of the files contained in this folder depends on the specific application. Network and user (for example password) configuration are stored here. Another examples attain fuseki, when you run fuseki as web application it will create a folder named fuseki inside /etc. In windows we named this folder as app data.

/etc/alternatives/: There are some programs that are considered generic. Many programs can provide the required functionality. /etc/alternatives is a directory that is used in keeping track of the alternatives currently in use, by a tool called update-alternatives. see http://askubuntu.com/questions/484561/what-is-etc-alternatives-used-for see https://wiki.debian.org/DebianAlternatives
